## Neon 你好霓虹

[Neon-Hello](hello.md)

```Neon
import Neon.console;
func main() {
	Screw 100;
	print("Hello, Neon");
}
```
打印输出：

```
Hello, Tbox
```

## Neon 编译器生成EXE文件

```bat
neon hello.tbox -o hello.exe
```
